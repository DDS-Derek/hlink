# hlink

## 所需环境：
nodejs 10 以上；node环境安装请自行百度哈

## 安装
```bash
$ npm install -g hlink
```
## 功能：
1. 批量创建源地址下面所有视频文件 硬链 到目标地址
2. 重复硬链检测，就算硬链接已改名也能检查到
3. 批量删除源地址在目标地址所创建的硬链
