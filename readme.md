# hlink
> 批量、快速硬链工具(The batch, fast hard link toolkit)

- 💡 重复检测：支持文件名变更的重复检测
- ⚡️ 快速：`20000+`文件只需要1分钟
- 📦 多平台：支持Windows、Mac、Linux
- 🛠️ 丰富的配置：支持黑白名单，缓存等多个配置
- 🔩 修剪机制：让你更方便的同步源文件和硬链

更多介绍：https://hlink.likun.me

## 安装
```bash
npm install -g hlink

# 查看帮助

hlink --help
```
