## 所需环境：
nodejs 12 以上；

## 安装
```bash
$ npm install -g hlink

# 查看帮助

$ hlink --help
```
## 与`cp -lr`的优势功能：
1. 重复硬链检测，就算硬链接已改名也能检查到(内容改了不行)
2. 更好的硬链删除机制
3. 更友好的的提示
